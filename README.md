# Joanna Joseph Portfolio

Production portfolio for Joanna Joseph, built with Next.js App Router, TypeScript, Styled Components, Storybook, Vitest, and jest-axe.

## Commands

```bash
npm run dev
npm run lint
npm run type-check
npm run test
npm run build
npm run build-storybook
```

## Notes

- `SPEC.md` is the source of truth for implementation decisions.
- `joanna-portfolio.html` is the visual reference.
- Do not add `babel.config.js`; Styled Components is enabled through SWC in `next.config.ts`.
